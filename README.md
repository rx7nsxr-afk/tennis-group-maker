# Tennis Group Maker (Vercel ready)

## 使い方
```bash
npm install
npm run dev
```

## Vercel
1. このフォルダを GitHub に push
2. Vercel で `New Project`
3. GitHub リポジトリを選択
4. Framework Preset は `Vite`
5. Build Command は `npm run build`
6. Output Directory は `dist`
7. Deploy
```

### ローカル確認
```bash
